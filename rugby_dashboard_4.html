⚠ Refresh werkt niet vanuit een lokaal bestand

Browsers blokkeren externe requests als het bestand geopend is via file://. Host het bestand om de verversknop te activeren:

  1. Ga naar app.netlify.com/drop
  2. Sleep dit HTML-bestand naar het venster
  3. Kopieer de URL die je krijgt (bijv. xyz.netlify.app)
  4. Sla die URL op als bladwijzer — refresh werkt daar wél
